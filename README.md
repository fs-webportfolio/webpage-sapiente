Dieses Projekt habe ich mit HTML3 und SCSS sowie einem kleinen JS für den Burger-Button umgesetzt. Dabei kommen maps, functions und mixins zum Einsatz. 

Die functions ermöglichen die automatische Kompilierung von css-Variablen aus einer Map für die gezielte Nutzung als Inline-Style.
Ebenfalls nutze ich eine brightness-function zur automatischen Erzeugung der Schriftfarbe des Buttons, sowie beim hover zur Berechnung der Abdunklung der Button-Farbe.

Ein @container-query nutze ich für gezielte Anpassung eines Containers für individuelle Steurung bei Display-Verkleinerung.
